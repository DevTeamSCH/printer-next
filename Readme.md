# Printer-next

## Requirements

- python 3.5+
- pip, pipenv
- npm

## Development
1. Install dependencies from the pipfile

2. Install node dependecies

```` npm install````

3. Setup environmental variables

```` cp example.env .env````

```` source .env ````

4. Migrate and run server

```` python3 -Wall manage.py migrate ````

```` python3 -Wall manage.py runserver ````
